Module TuringMachine.Tape

type 'symbol tape = 'symbol list * 'symbol * 'symbol list

Define a 3-tuple which represent a infinite tape.

The first list represent the left side. The second element represent the element of the head. The third represent the right side.

It's an infinite tape because the left and right side can grown as the user want, it's limited only because of the hardware.
