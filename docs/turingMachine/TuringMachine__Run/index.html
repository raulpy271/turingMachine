val step : ('symbol, 'state) TuringMachine.Data.turingMachineID -> ('symbol, 'state) TuringMachine.Data.turingMachine -> ('symbol, 'state) TuringMachine.Data.turingMachineID option

This function aims to represent the behavior of a given machine. So, it gives each step of the machine.

parameter id

This is the instantaneous description of a machine.

parameter machine

This is the machine which the id represents.

returns

The return is the next ID of the machine, this ID is given by the transition function. Besides, the returned value is inside a option because when the machine halt in the next step, it return None .
